# ClipFactory

Мобильное приложение для работы с видео и клипами.

## Требования к окружению
Для успешной сборки проекта через командную строку у вас должны быть установлены:
- **Java Development Kit (JDK)** версии 11 или выше.
- **Android SDK**.

## Инструкция по сборке APK через командную строку (CLI)

Сборка приложения осуществляется с помощью встроенного в проект Gradle Wrapper. **Открывать Android Studio для сборки не требуется.**

### Шаг 1. Клонирование репозитория
Откройте терминал (`cmd`, `PowerShell`, `Git Bash` или терминал Linux/macOS) и склонируйте проект:

```bash
git clone https://github.com/Yarrriiik/clipfarmlast.git
cd clipfarmlast
```

### Шаг 2. Запуск сборки
Находясь в корневой папке проекта, выполните команду для сборки отладочной (Debug) версии:

**Для Windows (cmd / PowerShell):**
```cmd
.\gradlew.bat assembleDebug
```

### Шаг 3. Расположение готового APK
После успешной сборки готовый установочный файл появится в директории проекта по следующему пути:

```text
app/build/outputs/apk/debug/app-debug.apk
```

### 🚀 Автоматическая установка через CLI (опционально)
Если к вашему компьютеру подключено Android-устройство по USB (с включенной «Отладкой по USB»), вы можете установить приложение прямо из терминала.

**Установка приложения на устройство (Windows):**
```cmd
.\gradlew.bat installDebug
```

### Запуск приложения
После завершения установки найдите иконку **ClipFactory** в меню всех приложений на вашем устройстве и запустите её.

## Руководство пользователя

### Вход в приложение
1. Откройте приложение.
2. Выполните вход или зарегистрируйте новый аккаунт.
3. После входа откроется экран проектов.

### Карта
1. Откройте раздел **Управление проектами**.
2. Выберите пункт **Выбор серверов**.
3. На экране откроется карта OpenStreetMap с метками наших серверов.
4. Нажмите на метку, чтобы увидеть её описание.

### Внешний API
1. На том же экране ниже карты отображается блок **Данные Open-Meteo**.
2. В блоке показываются текущая температура, скорость ветра и время наблюдения.
3. Для работы этого блока нужен доступ в интернет.

### Полезно знать
- Если карта или данные API не загрузились, проверьте интернет-соединение.
- Экран предназначен для демонстрации работы с картой и публичным API в рамках практической работы.
